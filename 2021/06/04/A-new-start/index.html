A new beginning!

發表於 2021-06-04 更新於 2021-06-08 分類於 Random Disqus：

此 Blog 是基於更早先在 Google blogger 上的版本 Sic Itur Ad Astra 而來，名字不變。不過作為一個新的開始，我並不打算將舊版本的文章移植過來，我認為他們更標誌著過去的里程碑，如果有興趣過去我寫了什麼，可以回到早先的 Google blogger 上去閱讀。在本次新的版本中將架於 github 上，文章發佈使用 hexo，數學的部分使用 MathJax 來支援 LaTeX\LaTeXLATE​X 語法。

這次新的遷移，寫作主旨不變，一樣會是我自己的閱讀心得，包含一般讀物，或是一些科學與技術文章，有些或許只是我自己的推導，作為一個理解。另外我也非常喜歡仿真模擬，如果可以用簡單的 script 來模擬物理世界發生的現象，對我來說會是非常有意思的。腳本的部分最常使用的會是 python，同 Google blog 的作法，numpy as np 一定是預設引入的，原始碼同樣會放在我的 github 上，一些比較簡單的我就不放了，但如果真的還是做不出文章內的結果又很有興致的話話就來信索取吧！
