亂數之母: 均勻分佈

發表於 2021-07-26 更新於 2021-08-03 分類於 Mathematics Disqus：

均勻分佈 (uniform distribution)，或稱勻分佈，他指涉的是在 [0,1) 之間均勻分佈的隨機數。幾乎所有語言的套件庫都會內建一個具備該功能的函式，例如 python 中標準的 random 模組中便由其中一個名為 random 函數來生成，當然更常使用的數值計算套件 numpy 中的同名 random 模組也有一個對應的 uniform 函數，他們做的都是相同的事情，至於其他套件或模組所自帶的生成在 [a,b) 間均勻分佈的隨機數已有太多，就不再贅述。但在統計學上，似乎我們更重視像高斯分佈、指數分佈或冪分佈等等分佈函數相關的隨機數而不是均勻分佈，例如我們會使用高斯分佈來描述平衡態系統的情況 (平衡態系統具有最大熵，可參閱舊文)，那為什麼我要專文談均勻分佈呢？因為在計算科學上，這些稀奇古怪的機率分佈所對應的隨機數，都可以通過均勻在 [0,1) 之間生成的隨機數變換而來，所以構造一個好的勻分佈演算法對於描述其他機率分佈的隨機變數至關重要。

探索 Monte Carlo 方法: Ising model revisit

發表於 2021-07-21 更新於 2021-08-08 分類於 Mathematics Disqus：

這篇文章裡我想要聊一聊模擬物理系統演化的一種方法，該演化的方法是基於機率採樣，亦即從所有可能演化的狀態中隨機選出其中一個作為下一步的狀態，這種具有隨機性的方式便稱做蒙地卡羅法 (Monte Carlo method)，或簡稱 MC。通過特定的條件限制 (constraints)，每種演化的情況都被賦予了特定的機率權重，而 MC 法則是通過採樣器去從多個可能的結果中採樣 (sampling) 出一個情況作為這一步結束後的狀態更新，所以即便是小概率的演化情況也是有可能被採樣器所取出。MC 基於這一特性，使它具有不易困在局域最佳解 (local optimization, local minimum)，而有機會將解收斂在全域最優 (global optimization, global minimum) 的優勢，相較於一些決定性演算法而言 (貪婪算法便不具備這一優勢，而有可能被困在局域最佳)。
